## Setup
```
  npm install
```

## Start
```
  npm start
```
It will start a server at port `3000` by default.

## Running tests

```sh
npm test
```

## Running lints

```sh
npm run lint
```

## Running types checks

```sh
npm run types:check
```

## Building app

```sh
npm run build
```
